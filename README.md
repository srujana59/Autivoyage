# Autivoyage – Interactive Learning Games for Autistic Kids

Autivoyage is a simple and interactive web-based gaming platform designed for autistic children. The project includes multiple educational and fun games that help improve memory, recognition, problem-solving, color matching, and logical thinking skills.

## Project Overview

This project provides a child-friendly interface where users can enter their name and access different learning games. Each game is designed with simple visuals, easy navigation, and engaging activities to support learning in a playful way.

## Features

* Simple and colorful user interface
* Name entry screen for personalized experience
* Multiple educational games
* Easy navigation between pages
* Browser-based project with no backend required
* Suitable for beginners to understand HTML, CSS, and JavaScript

## Games Included

### 1. Emotion Recognition Game

Helps children identify and understand different emotions using visual interaction.

### 2. Color Matching Game

Improves color recognition by matching colors with related objects or images.

### 3. Puzzle Game

Enhances problem-solving and logical thinking through puzzle-based activities.

### 4. Sudoku Game

A simple Sudoku-based activity that helps improve concentration and reasoning.

### 5. Memory Game

Improves memory power by matching similar cards.

## How to Run the Project

### Method 1: Open Directly

1. Download or extract the project ZIP file.
2. Open the project folder.
3. Go to the main project folder.
4. Double-click on:

```text
autiovoage.html
```

5. The project will open in your browser.

### Method 2: Using VS Code Live Server

1. Open the project folder in Visual Studio Code.
2. Install the **Live Server** extension.
3. Open `autiovoage.html`.
4. Right-click on the file.
5. Select **Open with Live Server**.

## Important Note

Make sure all HTML files are in the same folder or the navigation links may not work properly. Do not run the project directly from inside the ZIP file. Extract the ZIP first and then open the files.
